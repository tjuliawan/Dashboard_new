## 1. Ringkasan Pekerjaan

Hari ini fokus pada modul **Gudang**, meliputi:

1. Penyesuaian halaman **Rekap Stock Rack** agar sesuai dengan referensi C# `PrintPreviewReportRekapStockRack` (`grdD_format()`).
2. Penambahan / pengembalian kolom **Price, Value, dan 7 kolom COGS** pada Rekap Stock Rack.
3. Pembuatan menu & halaman baru **Price List** (dua panel: *Update by Supplier* dan *Price List / by SKU*).
4. Perombakan layout Price List dari 2 kolom (kiri/kanan) menjadi vertikal (atas/bawah).
5. Pengubahan panel **Update by Supplier** dari tampilan tabel menjadi **form input** untuk update harga.
6. Penambahan menu baru **Track In/Out** di bawah Price List.

## 2. Detail Perubahan per File

### 2.1 `app/Http/Controllers/GudangController.php`

- **`rekapStockRack()`**
  - Mengembalikan sub-aggregate COGS via `leftJoinSub` ke `ms_cogs`:
    - `ROW_NUMBER() OVER (PARTITION BY sku_business ORDER BY rec_datecreated DESC)` — ambil 1 baris terbaru per `sku_business`.
  - Menambah kolom: `price` (dari `SKU_hargabeli_pcs`), `value` (= `price × Stock_akhir`), serta 7 kolom COGS:
    `cogs_pcs`, `cogs_ctn`, `last_cogs_pcs`, `cogs_harga_terakhir_pcs`, `cogs_harga_terakhir_ctn`, `cogs_price_list_pcs`, `cogs_price_list_ctn`.
- **`priceList()` (baru)**
  - Mirror SP `Sp_List_tr_TGUPriceList_Supplier` (CTE row_number untuk harga terbaru & sebelumnya).
  - Sumber data: `tr_TGUPriceList` + `tr_TGUPricelist_d`, JOIN ke `ms_cabang`, `ms_supplier`, `TGU_ms_product_price`, `ms_unit`, `tgu_ms_product_internal`, `tgu_ms_product_Business`.
  - Helper privat: `priceListBySupplier()` dan `priceListBySku()`.
  - Kemudian disederhanakan: panel Supplier tidak lagi menampilkan tabel; hanya panel SKU yang query.
- **`priceListUpdate()` (baru)**
  - Endpoint `POST /gudang/price-list` untuk menyimpan harga baru.
  - Validasi: branch, supplier, sku_code, sku_supplier (required); price_pcs (required numeric); price_ctn (nullable).
  - Insert ke `tr_TGUPriceList` (header, kode `PL{YmdHis}`) + `tr_TGUPricelist_d` (detail) dalam `transaction()`.

### 2.2 `resources/views/gudang/rekap_stock_rack.blade.php`

- Header tabel: kolom **Perhitungan user** + 9 kolom baru (Price, Value, COGS Pcs/Ctn/Last/Harga Terakhir Pcs/Ctn/Price List Pcs/Ctn).
- Body `<td>`: menambahkan 9 cell terkait dengan `number_format(...)`.

### 2.3 `resources/views/gudang/price_list.blade.php` (baru)

- Layout responsif dengan CSS variables (`--pl-bg-page: #5c98fa`, dst.).
- **Layout final**: grid 1 kolom (panel ditumpuk vertikal).
- **Panel 1 — Update by Supplier**: form input (Branch, Supplier, SKU, SKU Supplier, Price PCS/CTN, Catatan) + tombol *Save Price*. Tidak ada tabel.
- **Panel 2 — Price List / by SKU**: filter (Branch, Supplier, SKU, Product, per_page) + tabel hasil + pagination.
- Flash message `pl_success` dan tampilan error validasi.

### 2.4 `routes/web.php`

```php
Route::get ('/gudang/rekap-stock-rack', [GudangController::class, 'rekapStockRack'])->name('gudang.rekap-stock-rack');
Route::get ('/gudang/price-list',       [GudangController::class, 'priceList'])->name('gudang.price-list');
Route::post('/gudang/price-list',       [GudangController::class, 'priceListUpdate'])->name('gudang.price-list.update');
```
