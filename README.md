# 📊 DN Tagih Management System

![Laravel](https://img.shields.io/badge/Laravel-11.x-red.svg)  ![PHP](https://img.shields.io/badge/PHP-8.2+-blue.svg)  ![License](https://img.shields.io/badge/License-MIT-green.svg)  ![Status](https://img.shields.io/badge/Status-Active-brightgreen.svg)

**DN Tagih Management System** is a web application built with **Laravel** designed to help businesses manage and monitor invoices based on sales data. This application uses the **Soft UI** template for a modern and responsive user interface.

---

## 🎯 Key Features

- ✅ Sales and invoice data management  
- ✅ User authentication and management  
- ✅ Dashboard with interactive statistics and graphs  
- ✅ Elegant UI design with **Soft UI Template**  
- ✅ Export invoice data to **PDF** / **Excel**  
- ✅ Dynamic search and filtering of invoice data  

---

## 🛠️ Technologies Used

- **Laravel 11.x**
- **PHP 8.2+**
- **MySQL** / **MariaDB** / **MS SQL Server**
- **Bootstrap** (using Soft UI Template)
- **jQuery & AJAX** for interactive features
- **Composer** & **NPM**
